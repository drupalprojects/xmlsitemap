INSTALLING XML SITEMAP
----------------------

1. Place the xmlsitemap directory in your modules directory (usually
   sites/all/modules).

2. Go to the XML sitemap section at
   http://www.example.com/?q=admin/build/modules and enable XML sitemap and any
   of the optional related modules you want to use.

   Optional modules included with XML sitemap:

   XML sitemap menu - Adds menu items to the site map.
   XML sitemap node - Adds nodes to the site map.
   XML sitemap: File - Adds node file attachments to the site map.
   XML sitemap term - Adds taxonomy terms to the site map.
   XML sitemap user - Adds user profiles to the site map.
   XML sitemap: Engines - Supports Google, Yahoo!, Ask, and Windows Live.

3. The sitemap is now operational at http://www.example.com/?q=sitemap.xml. See
   README.txt for configuration instructions.

UPGRADING FROM OLDER VERSIONS
-----------------------------

XML sitemap 5.x-1.x:
Install the latest 5.x-1.x release and run update.php before upgrading.

Google Sitemap:
Install the latest 5.x-1.x release of XML sitemap. To ensure that your Google
Sitemap settings are transferred correctly, enable XML sitemap before removing
gsitemap from your modules directory.

UNINSTALLATION
--------------

1. Go to http://www.example.com/?q=admin/build/modules and disable all XML
   Sitemap modules.

2. Go to http://www.example.com/?q=admin/build/modules/uninstall and uninstall
   all XML sitemap modules.
